# توقع مغادرة عملاء البنك باستخدام بايثون والتعلم الآلي

هذه نسخة عرض عامة للعميل من داشبورد توقع مغادرة عملاء البنك. تعرض النسخة بيانات تجريبية جاهزة، مؤشرات المخاطر، الرسوم البيانية، وجدول العملاء الأعلى خطورة، مع إمكانية تصدير تقارير تجريبية بصيغ Excel و PDF و CSV.

تم تعطيل زر استيراد ملفات العملاء في هذه النسخة العامة لحماية النموذج ومنع استخدامه على بيانات خارجية داخل معرض الأعمال. النسخة الكاملة الخاصة بالعميل يمكن أن تتضمن رفع ملفات CSV أو Excel وتشغيل النموذج على بياناته الفعلية.

## ما تعرضه النسخة

- إجمالي العملاء في بيانات العرض.
- عدد العملاء المتوقع مغادرتهم.
- عدد العملاء عاليي الخطورة.
- متوسط احتمالية المغادرة.
- توزيع العملاء حسب مستوى الخطر.
- متوسط الخطر حسب الدولة.
- متوسط الخطر حسب نشاط العميل.
- متوسط الخطر حسب عدد المنتجات.
- جدول العملاء الأعلى خطورة.
- تصدير تقارير تجريبية.
